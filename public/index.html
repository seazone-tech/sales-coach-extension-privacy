Como instalar

🪟 Windows (caminho rápido)

  1. Feche o Google Chrome (todas as janelas).
  2. Extraia o ZIP que você acabou de baixar.
  3. Abra a pasta Seazone-Sales-Coach.
  4. Dê duplo clique em INSTALAR.reg.
  5. Confirme as duas perguntas do Windows (a primeira é o Editor do Registro pedindo permissão).
  6. Abra o Chrome — em segundos a extensão será instalada automaticamente.
  7. Entre em uma reunião do Google Meet → a sidebar aparece no canto direito.
Smart App Control bloqueando? O .reg é seguro — ele só adiciona uma política do Chrome no registro do Windows, não executa programa. Pode confirmar com tranquilidade.

🍎 macOS

  1. Feche o Google Chrome (todas as janelas).
  2. Extraia o ZIP e abra a pasta Seazone-Sales-Coach.
  3. Abra o Terminal (Cmd+Espaço → "Terminal").
  4. Arraste o arquivo outros-sistemas/install-mac.sh para o Terminal e pressione Enter.
  5. Siga as instruções na tela.
  6. Abra o Chrome — a extensão será instalada.

🐧 Linux

  1. Feche o Google Chrome.
  2. Extraia o ZIP e abra um terminal na pasta Seazone-Sales-Coach.
  3. Execute: bash outros-sistemas/install-linux.sh
  4. Digite sua senha quando o sistema pedir.
  5. Abra o Chrome.

Não funcionou?

Antes de pedir ajuda, verifique:
  • Você fechou completamente o Chrome antes de instalar?
  • Abra chrome://policy no Chrome — deve aparecer ExtensionInstallForcelist. Se não aparecer, clique em "Recarregar políticas" no topo.
  • Como último recurso, reinicie o computador.

Se nada funcionar, fale com Luiz Lopes (Sales Ops).

Como desinstalar

No Windows, dê duplo clique em DESINSTALAR.reg. No macOS/Linux, rode outros-sistemas/uninstall-mac.sh ou outros-sistemas/uninstall-linux.sh.
